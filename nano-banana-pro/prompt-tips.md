# Prompt Engineering Tips for Nano Banana Pro

Share these with the user when they're crafting or refining prompts.

## General Tips

- **Be specific**: "A golden retriever puppy playing in autumn leaves, soft afternoon light, shallow depth of field" beats "a dog"
- **Specify style**: "watercolor painting", "3D render", "photorealistic", "flat vector illustration", "oil painting", "pixel art"
- **Include lighting**: "dramatic side lighting", "soft diffused studio light", "golden hour", "neon glow"
- **Mention composition**: "close-up", "bird's eye view", "rule of thirds", "centered", "wide angle"
- **Specify mood**: "moody", "cheerful", "minimalist", "cinematic", "ethereal"

## Text in Images

Nano Banana Pro has industry-leading text rendering (~94% accuracy, multilingual):

- Put text in quotes: `A neon sign that says "OPEN 24/7"`
- Specify font style: `bold serif font`, `handwritten script`, `monospace`
- Works in multiple languages

## Aspect Ratio Guide

| Use case | Aspect ratio |
|---|---|
| Social media square | 1:1 |
| YouTube thumbnail | 16:9 |
| Phone wallpaper | 9:16 |
| Instagram story | 9:16 |
| Portrait photo | 3:4 or 4:5 |
| Landscape photo | 3:2 or 16:9 |
| Ultra-wide banner | 21:9 |
| Blog header | 16:9 |
| Book cover | 2:3 |

## Editing Prompts

When editing existing images, be precise about what to change and what to keep:

- Good: "Change the sky to a dramatic sunset while keeping the foreground unchanged"
- Good: "Add a subtle vignette effect and increase the warmth of the lighting"
- Bad: "Make it better" (too vague — the model may change everything)

## Iteration Strategy

When iteratively refining in a session:

1. Start with broad strokes: overall composition, subject, style
2. Then refine details: lighting, colors, specific elements
3. Final pass: text placement, fine details, polish
